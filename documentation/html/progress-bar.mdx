# Tailwind CSS Progress Bar

Our Tailwind CSS progress bar component can be used to show a user how far along he is in a process. The progress can be determinate or indeterminate. 

Use the Progress Bar to show an ongoing process that takes a noticeable time to finish. It helps users estimate how much time an operation might take or simply reassuring them that the system is actively working on something.

Below we are presenting our examples of progress components that you can use in your Tailwind CSS project. They come in different styles so you can adapt them easily to your needs.

<br />

## Progress Bar Best Practices for Developers

• Provide feedback when the process is complete (message, sound). <br />
• Use progress bars only when necessary - not every loading or processing action requires a progress bar.<br />
• Indeterminate progress bars should be used sparingly, only when the duration cannot be determined.<br />
• Make sure the progress bar accurately reflects the current status of the process it represents.<br />
